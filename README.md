# Proyecto 3: Space Travel

**dependencias utilizadas:**
* nalgebra-glm
* minifb
* image
* rayon

**Como movilizarse:**
1. Adelante/Atras/Izquierda/Derecha: W / S / A / D
2. Arriba/Abajo: Shift Izquierdo / Espacio
3. Rotación: ← / → / ↑ / ↓
4. Zoom: Q / E
*Cambio de vista (1ra y 3ra)*: V

> Nota: Se inicia en primera persona. En tercera persona se ve la nave
